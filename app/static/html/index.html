AI Chatbot for Academic Assistance

An intelligent academic assistant powered by LLMs, Retrieval-Augmented Generation (RAG), Hybrid Search, and Local AI Models.

Start Chatting Explore Features

Features

PDF Question Answering

Upload academic PDFs and ask contextual questions directly from documents.

Hybrid Search

Combines BM25 keyword search with semantic vector retrieval for higher accuracy.

Conversation Memory

Maintains chat history and contextual continuity across conversations.

Local LLM Support

Uses Ollama to run AI models locally without external API costs.

FastAPI Backend

Async backend architecture designed for scalability and performance.

Vector Database

ChromaDB stores embeddings for fast semantic document retrieval.
